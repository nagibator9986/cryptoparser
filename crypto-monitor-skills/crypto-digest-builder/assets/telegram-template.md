# Telegram MarkdownV2 — шаблон сегментов сводки

## Важно об экранировании

В Telegram MarkdownV2 ОБЯЗАТЕЛЬНО экранировать следующие символы обратным слэшем:
```
_ * [ ] ( ) ~ ` > # + - = | { } . !
```

Это значит, что в готовом тексте сводки **все точки, дефисы, скобки экранируются**: `12\.03`, `--\->`, `\(SEC\)`.

При генерации сегментов применяй экранирование автоматически КО ВСЕМУ ТЕКСТУ кроме разметки.

## Структура первого сегмента (шапка + первый раздел)

```
📊 *Ежедневная сводка*
*Цифровые активы и криптоиндустрия*

📅 12 марта 2025 \| 22 публикации

\=\=\=

🇰🇿 *Регулирование Республики Казахстан*

🟠 *HIGH*
*Запуск второй фазы пилота цифрового тенге с участием четырёх банков*

Национальный банк РК объявил о начале второй фазы пилотного проекта цифрового тенге\. К проекту подключились четыре системообразующих банка — Halyk, Kaspi, Forte и Jusan\.\.\.

📰 nationalbank\.kz \| 📅 12\.03 \| [Читать оригинал](https://nationalbank.kz/news/123)

\-\-\-

🟡 *MEDIUM*
*AFSA выдало лицензию криптопровайдеру XYZ Crypto*

Агентство по регулированию финансовых услуг МФЦА выдало лицензию компании XYZ Crypto на оказание услуг хранения цифровых активов\.\.\.

📰 afsa\.aifc\.kz \| 📅 12\.03 \| [Читать оригинал](https://afsa.aifc.kz/news/456)

\-\-\-

_продолжение далее\.\.\._
```

## Структура промежуточного сегмента

```
\(продолжение\)

🌍 *CBDC и государственные цифровые инициативы*

🟠 *HIGH*
*Заголовок второй новости*

Текст реферата\.\.\.

📰 источник \| 📅 12\.03 \| [Читать оригинал](URL)

\-\-\-

_продолжение далее\.\.\._
```

## Структура последнего сегмента

```
\(окончание\)

🌐 *Кратко: международные новости*

🟡 *MEDIUM*
*Заголовок*

Текст\.\.\.

📰 источник \| 📅 12\.03 \| [Читать оригинал](URL)

\=\=\=

_Сводка сформирована автоматически\. Архив: [тык](https://archive.example.kz)_
```

## Эмодзи для разделов

| Раздел | Эмодзи |
|--------|--------|
| Регулирование РК | 🇰🇿 |
| Регулирование СНГ и ЦА | 🌐 |
| CBDC и гос. инициативы | 🏛️ |
| Банки и финтех | 🏦 |
| Биржи, продукты, лицензирование | 💱 |
| Технологии, инфраструктура, безопасность | ⚙️ |
| Кратко: международные новости | 🌍 |

## Эмодзи для приоритетов

| Приоритет | Эмодзи + текст |
|-----------|----------------|
| critical | 🔴 *CRITICAL* |
| high | 🟠 *HIGH* |
| medium | 🟡 *MEDIUM* |
| low | ⚪ *LOW* (обычно не выводится) |

## Лимиты

- **Один сегмент:** не более 4000 символов (с запасом до лимита 4096 Telegram).
- **Не разрывай блок публикации** между сегментами.
- **Если один блок длиннее 4000 символов** (что не должно случаться при объёме реферата 60–120 слов) — сокращай реферат на лету.

## Алгоритм формирования сегментов

1. Собери полный текст всех блоков с разделителями.
2. Считай длину текущего сегмента (помни про дополнительные ~100 символов на «продолжение далее»).
3. Перед добавлением нового блока проверь: войдёт ли он в текущий сегмент?
4. Если не входит — закрой текущий сегмент маркером `_продолжение далее..._` и начни новый с маркером `(продолжение)`.
5. В последнем сегменте — закрывающий разделитель и подвал.
