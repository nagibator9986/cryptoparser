# Style guide — стилевой регламент сводки

## Принципы

Сводка — корпоративный документ, который читают руководители и аналитики банка. Тон — деловой, без сенсаций, без эмоциональных оценок. Сводка должна быть лаконичной, информативной, читаемой за 3–5 минут.

## Что недопустимо

- Кликбейтные заголовки («Шокирующее решение», «Прорыв года»).
- Эмодзи в теле текста (только в маркерах разделов и приоритетов).
- Сленг и жаргон («крипта», «бычий тренд», «луна»).
- Сравнения с предыдущими событиями без подкрепления данными.
- Указание имён участников без должностей и принадлежности.
- Прогнозы и оценки авторства самой сводки.

## Что обязательно

- Дата публикации каждой новости.
- Активная ссылка на оригинал.
- Имя источника (домен или название).
- Маркер приоритета.
- Чёткая структура: заголовок — реферат — атрибуция.

## Длина

| Элемент | Целевой объём |
|---------|---------------|
| Заголовок раздела | 3–6 слов |
| Заголовок публикации (title_ru) | 50–100 символов |
| Реферат (summary) | 60–120 слов |
| Атрибуция | 1 строка |
| Шапка сводки | 2–3 строки |
| Подвал | 2–4 строки |

## Тональность по разделам

| Раздел | Тон |
|--------|-----|
| Регулирование РК | Особо нейтральный; это локальный контекст, риск интерпретаций минимизировать |
| Регулирование СНГ | Нейтральный |
| CBDC | Нейтральный, акцент на технические факты |
| Банки и финтех | Деловой; акцент на бизнес-факты (партнёрства, продукты) |
| Биржи и продукты | Деловой |
| Технологии | Технически-нейтральный |
| Международные | Краткий обзорный |

## Сокращения и форматирование

- Денежные суммы: «$1,3 млрд», «120 млн тенге», «€500 млн».
- Большие числа: «10 000 пользователей», «1,5 миллиона транзакций».
- Даты в атрибуции: «12.03» (день и месяц через точку).
- Время: «09:00 по Алматы», «14:00 UTC».
- Аббревиатуры — без точек: SEC, MAS, CBDC.
- Знак рубля и тенге — словом («рублей», «тенге»), не символом.

## Цветовая палитра приоритетов

| Приоритет | HEX | Эмодзи | Использование |
|-----------|-----|--------|---------------|
| critical | #D32F2F | 🔴 | События критической значимости (см. prioritizer) |
| high | #ED6C02 | 🟠 | Значимые события |
| medium | #1976D2 | 🟡 | Обычные значимые события |
| low | #66686B | ⚪ | Малозначимые (обычно не выводятся) |

## Брендовая палитра

- Основной синий (заголовки, шапка): #1F4E79
- Акцентный синий (ссылки): #2E75B6
- Светлый фон карточки: #F5F9FD
- Текст: #1F1F1F
- Серый подвал: #66686B
- Фон страницы (email): #F4F6F8
- Линии-разделители: #E4E7EB

## Шрифты

- Email: Arial (универсальный, поддерживается всеми клиентами).
- Telegram: системный (определяется клиентом).
- Размеры email:
  - Шапка H1: 22px
  - Заголовок раздела H2: 16px
  - Заголовок публикации H3: 15px
  - Текст реферата: 13px
  - Атрибуция: 11px
  - Подвал: 11px
