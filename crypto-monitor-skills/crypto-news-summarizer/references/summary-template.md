# Summary template — структурный шаблон реферата

## Идеальная структура реферата (80–100 слов)

Реферат состоит из 4–6 предложений в одном-двух абзацах. Шаблон ниже — ориентир, а не жёсткая форма; следуй ему по духу, а не по букве.

```
[Предложение 1: ЧТО произошло]
Главный субъект + действие + объект. Без вводных слов.

[Предложение 2: КТО участники и существенные детали]
Кто ещё вовлечён, в какой роли, какие ключевые сущности упомянуты.

[Предложение 3: КОГДА и ГДЕ]
Дата, период, страна, юрисдикция (если не очевидно из контекста).

[Предложение 4: КЛЮЧЕВЫЕ ЦИФРЫ]
Суммы, объёмы, процентные показатели, количества.

[Предложение 5: ЗНАЧЕНИЕ / КОНТЕКСТ (опционально)]
Только если в источнике явно сказано. Никаких собственных прогнозов.

[Предложение 6: ПРЕДУПРЕЖДЕНИЕ (опционально)]
Если данные неподтверждённые или противоречивые — атрибуция.
```

## Примеры структур по типам событий

### Регуляторное действие

```
1. Регулятор X объявил о Y в отношении компании Z.
2. По данным регулятора, нарушение состоит в W.
3. Решение принято [дата] на территории [юрисдикция].
4. Размер штрафа / срок действия — N.
5. Компания может обжаловать решение / уже заявила позицию.
```

### Запуск продукта

```
1. Компания X запустила продукт Y для аудитории Z.
2. Особенности продукта: ключевые функции.
3. Запуск состоялся [дата] в [страны/регионы].
4. Целевой объём / стоимость / условия — N.
5. Контекст: место продукта в стратегии компании (если упомянуто).
```

### CBDC / государственный пилот

```
1. Центральный банк X объявил о фазе Y пилота CBDC.
2. К пилоту присоединились участники: P1, P2, P3.
3. Период пилота: с [дата1] по [дата2].
4. Целевые показатели: N пользователей, M транзакций.
5. Особое внимание: [техническая фокусировка].
```

### Биржевой инцидент

```
1. Биржа X сообщила о Y / На бирже X произошло Y.
2. Затронуты: N пользователей / M средств.
3. Время события: [дата/период].
4. Реакция биржи / регулятора.
5. (Если применимо) Предупреждение: «по неподтверждённым данным».
```

## Чек-лист после написания

- [ ] 60–120 слов (идеал 80–100)?
- [ ] Все ключевые сущности упомянуты?
- [ ] Все существенные цифры включены?
- [ ] Нейтральный тон без оценочных эпитетов?
- [ ] Нет цитат длиннее 15 слов?
- [ ] Нет «прогнозов» и «выводов» автора реферата?
- [ ] title_ru информативный, до 100 символов?
